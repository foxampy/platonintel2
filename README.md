## 🎯 Обзор проекта

**Plato-Intel v2** — современный корпоративный веб-сайт для поставщика электротехнического оборудования. Проект разработан с акцентом на:

- **Производительность** — оптимизированная загрузка страниц
- **SEO** — семантическая разметка, микроразметка Schema.org
- **Адаптивность** — корректное отображение на всех устройствах
- **Доступность** — соответствие стандартам WCAG 2.1
- **Интерактивность** — плавные анимации и современный UX

### Ключевые функции

- 📦 Каталог продукции с фильтрацией и поиском
- 🔍 Детальные страницы товаров с техническими характеристиками
- 📱 Полная адаптивность для мобильных устройств
- 🌐 SEO-оптимизация и мета-теги для всех страниц
- 🎨 Современный индустриальный дизайн
- ⚡ Оптимизированная производительность

## 🚀 Начало работы

### Требования

- Любой современный веб-браузер
- Локальный сервер (для полноценной работы с JSON)

### Установка

1. **Клонирование репозитория:**
```bash
git clone https://github.com/platointel/website.git
cd website/v2
```

2. **Запуск локального сервера:**

Используя Python:
```bash
# Python 3
python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Используя Node.js:
```bash
npx serve
```

Используя PHP:
```bash
php -S localhost:8000
```

3. **Открытие в браузере:**
```
http://localhost:8000
```

### Развёртывание на продакшн

1. Загрузите все файлы на веб-сервер (Apache/Nginx)
2. Убедитесь, что включён модуль `mod_rewrite` (Apache)
3. Настройте сжатие Gzip в `.htaccess`
4. Проверьте права доступа к файлам (644) и папкам (755)

## 💻 Разработка

### Код-стайл

- Mobile-first подход

**JavaScript:**
- ES6+ синтаксис
- Модули
- Async/await для асинхронных операций

### Добавление нового товара

1. Откройте `data/products.json`
2. Найдите нужную категорию
3. Добавьте объект товара:

```json
{
  "id": "product-id",
  "name": "Название товара",
  "slug": "product-slug",
  "category": "category-id",
  "price": 10000,
  "currency": "RUB",
  "availability": "in_stock",
  "stock_quantity": 10,
  "image": "path/to/image.jpg",
  "short_description": "Краткое описание",
  "description": "Полное описание",
  "specifications": {
    "Параметр": "Значение"
  }
}
```

### Добавление новой страницы

1. Создайте HTML-файл в корне
2. Подключите необходимые стили:
   ```html
   <link rel="stylesheet" href="css/main.css">
   <link rel="stylesheet" href="css/animations.css">
   ```
3. Добавьте meta-теги для SEO
4. Обновите sitemap.xml
5. Добавьте ссылки в навигацию
