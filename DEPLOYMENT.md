### 3. Аналитические системы
✅ **Создана система аналитики** (`js/analytics.js`)

**Поддерживаемые системы:**
- ✅ Яндекс.Метрика (требуется ID)
- ✅ Google Analytics 4 (требуется ID)
- ✅ Внутренняя аналитика (localStorage)

**Отслеживаемые события:**
- Просмотры страниц
- Просмотры товаров
- Добавление в корзину/запрос
- Клики по телефонам
- Клики по email
- Открытие чата
- Отправка сообщений в чат
- Глубина скролла (25%, 50%, 75%, 90%, 100%)
- Время на странице
- Исходящие ссылки

**Настройка:**
```javascript
// В js/analytics.js замените:
const CONFIG = {
    yandex: {
        enabled: true,
        id: 'YOUR_YANDEX_METRIKA_ID'  // ← Заменить
    },
    google: {
        enabled: true,
        id: 'G-YOUR_GA4_ID'  // ← Заменить
    }
};
```

**Пример обработки формы:**
```javascript
// Добавьте в конец forms.js для отправки на email:
document.getElementById('contactForm').addEventListener('submit', async (e) => {
    e.preventDefault();
    const formData = new FormData(e.target);
    
    // Отправка через EmailJS или Formspree
    await fetch('https://formspree.io/f/YOUR_FORM_ID', {
        method: 'POST',
        body: formData
```
